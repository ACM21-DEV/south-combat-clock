# South Combat Clock

Event timer application for amateur combat sports tournaments.

## Quick Start

1. Extract the folder
2. Open `index.html` (double-click)
3. Press **F** for fullscreen

## Controls

| Action | Key/Button |
|--------|-----------|
| Start / Pause | Spacebar |
| Next Match | N |
| Reset | R |
| Match List | P |
| Fullscreen | F |
| Sound | 🔊 Button |

## Features

✅ **Match Clock**
- Countdown timer for each round
- Multi-round support with automatic rest counter
- Extra time option (1 minute)

✅ **Match Card (Cartelera)**
- Pre-loaded 9 matches
- Edit fighter names, type, duration, rounds anytime
- Add new matches with form
- Track results (Blue win / Red win / Tie)

✅ **Audio**
- Bell at round start and end
- Countdown beeps in final 10 seconds

✅ **Backup**
- Copy all matches as backup
- Paste to restore if needed

## File Structure

```
south-combat-clock/
├── index.html          ← Open this
├── fighters.js         ← Match data (editable)
├── images/             ← Fighter photos
└── sounds/             ← Bells (bell-start.mp3, bell-end.mp3)
```

## That's it

Open `index.html` and start timing matches.
